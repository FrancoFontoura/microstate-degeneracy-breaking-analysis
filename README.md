# microstate-degeneracy-breaking-analysis
This project investigates preliminary statistics of recurrence microstate degeneracy patterns under different dynamical regimes.

- Recurrence Analysis
- Time Series Analysis
- Statistical Modeling
- Dynamical Systems

## Methods

- Microstate sampling
- Statistical symmetry analysis
- Numerical simulations
- Graphical analysis

## Main Technologies

- Python
- NumPy
- SciPy
- Matplotlib
